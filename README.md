nagnu
=====

Nagios frontend

### Tested with
* Nagios 3.2.3
* Nagios 3.5.1
* Nagios 4.0.8

### Tested on
* MacOS X 10.9
* FreeBSD 10
* Scientific Linux 6.5

### Configuration
-c [configfile]

See nagnu.conf.sample

### Excludes
-e [excludesfile]

Place service_description in a file, one per line, to exclude them from the output.
